# springboot-mongodb

## 简介
---
基于spring-boot和mongodb写的一套基础框架，框架可用于支撑移动应用后台服务，接口是restful风格的。

## 使用方法
使用intellij maven导入
启动类 springboot-mongodb/src/main/java/com/lin/Application.java main 方法运行即可启动

启动后访问：restfull api
http://localhost:8080/api/v1.0/category
启动后访问：hello world
http://localhost:8080/index
